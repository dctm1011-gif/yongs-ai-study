개발방향정리

취향 분석 → 논문 추천 → 분류 추가 → 개별 HTML 생성 → 링크 업데이트까지 모두 자동화돼요.

알고리즘 제약 · 주의사항
arXiv HTML 버전: 일부 논문은 arxiv.org/html/이 404이에요. 그 경우 arxiv.org/abs/ abstract 페이지로 폴백해요.

Semantic Scholar / IEEE: 크롤링 제한이 있어 본문이 비어올 수 있어요. 이 경우 Claude의 사전학습 지식 + 기존 논문 분류 내용을 조합해서 분석해요.

코드워드 / 수식: HTML 파일 크기를 줄이기 위해 이미지는 포함하지 않고 텍스트와 .stat-box로 수치를 표현해요. 논문 원문 링크로 원문 확인이 가능해요.

history/index.html 7.5 MB 제한: col-card 블록이 지나치게 길어지면 per-paper HTML 분리 전략을 기본으로 유지해요.

취향 분석 업데이트: 새 논문이 추가될 때마다 위 취향 프로파일이 자동 갱신돼야 해요. 새 카테고리가 3회 이상 반복되면 pref-grid에 반영해요.
